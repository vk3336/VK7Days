# How to Fix Notification Issues

## Problem: "Enable alerts" shows "BLOCKED"

This happens when notification permission was previously denied.

## Solution:

### For Chrome/Edge:

1. **Click the ðŸ”’ lock icon** in the address bar (left of URL)
2. **Find "Notifications"** in the dropdown
3. **Change from "Block" to "Allow"**
4. **Refresh the page**
5. **Click "Enable alerts"** again

### For Firefox:

1. **Click the shield icon** in address bar
2. **Click "Permissions"**
3. **Find "Notifications"** and set to "Allow"
4. **Refresh the page**

### For Safari:

1. **Go to Safari > Settings > Websites**
2. **Find "Notifications"**
3. **Find your site and set to "Allow"**
4. **Refresh the page**
